# FLATROCK PROJECT FOR TESTING
> Reports & Forecasts using python, mongo and docker compose

## Used Technologies and methodologies
* Python
    * Flask(Flask, Flask-pymongo, Flask_wtf)
    * wtforms
    * Pandas
    * Pymongo
    * Numpy
* Jinja 2
* Mongo
* Docker & docker compose

## Usage

```bash
docker-compose build
docker-compose up
```
